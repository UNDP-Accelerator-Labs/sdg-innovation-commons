## 

Finalista del RETO DE INNOVACIÓN para crear un registro de jóvenes en informalidad lanzado en el marco del Proyecto SDG/ Laboratorio Aceleración Ecuador.


Es un prototipo para un sistema inteligente que permite recopilar datos de informalidad laboral para mejorar las oportunidades laborales de los jóvenes. Este sistema está dividido en cuatro partes importantes:


1. PROGRESSIVE WEB APP (PWA):
Esta aplicación ha sido diseñada para que los jóvenes tengan un espacio idóneo y muestren sus habilidades y conocimientos de su área de experticia al mundo. Todo esto acompañado de un asistente inteligente que brinda ofertas laborales ajustadas a cada uno de los perfiles en el caso que se requiera. Esta solución no solo ha sido

pensada para que los jóvenes encuentren trabajo sino que ellos puedan capacitarse día a día y estén preparados para el mundo laboral. Cuenta con una sección para que los jóvenes se capaciten de acuerdo a sus preferencias a través de cursos, blogs, consejos y post informativos para mejorar tanto sus habilidades técnicas como blandas.


La aplicación de DANIK cuenta con un sistema de PUSH NOTIFICATIONS automático para mantener al usuario informado y fidelizado con la plataforma. El sistema de notificaciones permite al usuario mantenerlo informado de noticias, ofertas laborales, eventos o consejos que le puedan servir en el día a día. Además, este sistema tiene la facultad de avisar cuando un nuevo cuestionario se ha creado, así el usuario estará informado en cualquier momento y la tasa de respuesta aumentará. El sistema de CORREO ELECTRÓNICO tiene el mismo fin mencionado
anteriormente pero se limita a informar sobre nuevos cuestionarios disponibles y noticias de carácter importante a la comunidad.


2. PANEL DE ADMINISTRACIÓN:
Este panel incluye:

· Crear, editar y publicar contenido informativo de manera sencilla.
· Manejo y administración de la página informativa (Landing Page).
· Manejo del sistema de PUSH NOTIFICATIONS.
· Estadísticas de la interacción con aplicación de los jóvenes (Rendimiento).
· Crear, editar y publicar encuestas fácilmente y en cualquier momento.
· Administración de la plataforma a través de roles de participación.


3. PANEL DE RECLUTAMIENTO:

Características destacadas:
· Filtro inteligente para buscar profesionales y no profesionales
· Comunicación directa con los jóvenes.
· Interfaz de usuario intuitiva para observar los portafolios de los jóvenes.
· Perfiles destacados.


4. DASHBOARD ESTADÍSTICO Y ACCESO A DATA WAREHOUSE

Características destacadas:
· Información relevante hallada a través de metodologías de Inteligencia deNegocios.
· Disponibilidad a la información 24/7.
· Facilidad de acceso a través de un sistema seguro (Contraseñas).
· Acceso a datos consolidados y homogéneos de forma segura a través de una API.
· Base de datos confiable para aplicar técnicas de analytics.
