## Description

> Add the description here

La solution est une application) propose une fonction de détection d'arbres qui permet aux utilisateurs d'obtenir des informations sur les plantes présentes dans une forêt à partir de flux d'images et de vidéos. En utilisant ce flux vidéo, le décompte des arbres offre une estimation du nombre approximatif d'arbres dans la forêt.
