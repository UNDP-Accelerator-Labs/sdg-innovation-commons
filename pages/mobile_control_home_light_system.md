The system can be able to live a comfortable life for physically disabled persons and save a lot of money which spent on switch installation and switch maintenance. Create a better living environment for physically disabled persons and save money which spent on switch installation and maintenance.
I have developed a mobile application and light bulb they can integrate with a wireless connection by Bluetooth so the user can make turn On and Off their home light system using their phone form a maximum of 15 meter distance between the lamb and their phone.
The technology will be able to save money, energy and create a better living environment for disable persons and other physically healthy customers too.
