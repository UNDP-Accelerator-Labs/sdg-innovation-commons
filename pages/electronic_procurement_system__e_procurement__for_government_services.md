The system includes a web application and mobile application platforms. The web application will serve both the government agencies and the bidder companies/suppliers. The mobile application will be used mainly by bidder companies/suppliers to apply bid. The government agency’s page includes creating procurement, tender announcement, tender evaluation and award, Procurement tracking, delivery confirmation and payment”

 Key features

 · Covers all functions in procurement, Pre-Tendering  Tendering and  Post-Tendering   

· Call for tender

· Check submitted bids

· Evaluate and award bids 

· Perform procurement agreement

· Track procurement, confirm delivery and pay for the goods
