X-ray and CT scan for testing COVID

> Solution description




●A web platform used to detect
COVID-19 by X-Ray and CT-Scan images
●The model is developed  using machine learning algorithm
●Fast response with less
contamination or physical contacts
●It uses available X-rays and CT
scan machines rather than specialized testing tool
●It works in parallel with the
current testing method of RT-PCR
