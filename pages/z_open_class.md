> Brief description of the innovation

Z open class is a web service software which helps educational institutions  and private teachers to distribute educational lessons to students across the world.

An institution can sign up and registister with zopen class web service to start distributing its lessons.

This app is to help institutions such colleges, universities and company to offer quality online courses to students
