Smart app that provides instant connection to the security when emergency arises. The app allows for registration to any resident association. Notifications are sent to security guards as well as other residents with the exact location of the emergency. It also allows for other function like visitor pas using their national ID to prevent unidentified strangers from entering the vicinity. It also allows for notification on community activities and events.
