> Description of the solution

MOOC is an online learning platform that is designed to be accessible to a large number of learners via the Internet. It aims to provide high - quality educational content in China. In this solution, it provides a digital agriculture course named 'Remote Sensing Digital Image Processing and Agricultural Applications'. This course is a core one for majors like Geographic Information Science. It focuses on three main aspects: preprocessing of remote sensing images, image enhancement and fusion, and agricultural applications.
